CoPerception-UAV+ & OPV2V+

Extended Virtual Collaborative Perception Dataset for UAVs and Autonomous Driving Vehicles.


About CoPerception-UAV+

The original CoPerception-UAVs is a large-scale UAV-based collaborative perception dataset, co-simulated by AirSim and CARLA. Here we introduce an extended version, CoPerception-UAVs+, which includes more collaborative agents (about 10). Each agent equips with 1 camera and 1 depth sensor with resolution 450 X 800. The detection range is 192m X 96m.



About OPV2V+

The original OPV2V is a large-scale vehicle-to-vehicle collaborative perception dataset, co-simulated by OpenCDA and CARLA. Here we introduce an extended version, OPV2V+, which includes more collaborative agents (about 10). Each agent equips with 4 cameras and 4 depth sensors with resolution 600 X 800. The detection range is 281.6m X 80m.
